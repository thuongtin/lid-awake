# Code of Conduct

## Our Standard

Lid Awake is a small technical project that touches local macOS power behavior. Contributors and maintainers are expected to keep discussion respectful, specific, and focused on making the software safer and clearer.

Examples of expected behavior:

- Use welcoming and inclusive language.
- Respect different experience levels and hardware setups.
- Give actionable technical feedback.
- Keep disagreements focused on code, behavior, documentation, and evidence.
- Avoid publishing private data, credentials, logs with personal information, or exploit details in public threads.

Examples of unacceptable behavior:

- Harassment, insults, threats, or discriminatory language.
- Personal attacks or sustained off-topic disruption.
- Publishing someone else's private information.
- Encouraging unsafe power-management behavior without clear warnings.
- Posting security exploit details outside the private reporting path in `SECURITY.md`.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

For conduct concerns, email the maintainer at `thuongtin@gmail.com`.

## Attribution

This code of conduct is intentionally short and project-specific. It is inspired by common Open Source community standards while keeping the enforcement process lightweight for a small maintainer-led project.
